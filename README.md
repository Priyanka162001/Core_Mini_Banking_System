# Savings Product Management API

## **Project Description**
This is a Spring Boot REST API for managing Savings Products in a banking system.

It allows:
- Create savings product
- Get product by ID
- Get all products
- Update product
- Delete product

## **Technologies Used**
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSql
- Maven

## **Security**
Role-based access:
- ADMIN → Create, Update, Delete
